# squiggs
My entry for squiggs!


### Installation

Clone the repo:
```
git clone https://github.com/darkystacks/squiggs
```

### Start

Enter the directory, run the commands below in your terminal, follow the local address(it will be right in terminal)
```
npm i
```
```
npm run build
```
```
npm run start
```
